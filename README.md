MVP версия учётной системы. 
Внедрены справочники, документы, регистры и вспомогательные объекты.
Внедрен оперативный учет с контролем остатков.В качестве партии принимается срок годности товара. Списание товаров реализовано в разрезе партий по методу FEFO (First Expire First Out, первый истекает — первым уходит), при котором сначала списываются товары с наименьшим сроком годности. Реализовано решение проблемы копеек и списание регистров в "0". (код можно посмотреть в модуле объекта РТУ)
